# 个体 T1 配准流程

## 什么时候需要配准

`T1图像` 目录中的 IXI 文件是原始个体空间，不能直接叠加内置的仿真图或 AAL 图谱。

要让图层重合，需要先把“模板空间图层”变换到“个体 T1 空间”。本目录的脚本执行的是：

```text
内置模板 T1  --配准到-->  个体 T1
内置模板空间仿真图  --同一变换重采样-->  个体 T1 空间仿真图
```

输出图层会使用个体 T1 作为 reference，因此 shape、spacing、origin/direction 与个体 T1 一致。

## 安装依赖

在用于运行脚本的 Python 环境中安装 SimpleITK：

```powershell
py -3 -m pip install SimpleITK
```

如果你的打包环境不是 `py -3`，请换成实际打包软件使用的 Python。

## 示例命令

下面示例把 ADHD/Left_DLPFC 的灰质、白质 subject 空间仿真层配准到 `IXI002` 个体 T1：

```powershell
cd /d "E:\卷在哪里\生医工竞赛\code_revise"

py -3 tools\register_template_to_individual.py `
  --fixed-t1 "E:\卷在哪里\生医工竞赛\T1图像\IXI002-Guys-0828-T1.nii.gz" `
  --template-t1 "mri\T1.nii" `
  --overlay "sample_data\ADHD\Left_DLPFC\grey_flex_mean_max_TI_mapped_TI_subject_TI_max.nii.gz" `
  --overlay "sample_data\ADHD\Left_DLPFC\white_flex_mean_max_TI_mapped_TI_subject_TI_max.nii.gz" `
  --out-dir "registered\IXI002_ADHD_Left_DLPFC"
```

完成后，在软件里：

1. 先加载原始个体 T1：`E:\卷在哪里\生医工竞赛\T1图像\IXI002-Guys-0828-T1.nii.gz`
2. 再手动追加 `registered\IXI002_ADHD_Left_DLPFC` 中生成的 `*_in_individual_space.nii.gz`

## AAL 和标签图谱

AAL、脑区标签、二值 mask 必须使用最近邻插值。脚本会自动按文件名识别 `aal/label/atlas/mask`，也可以显式指定：

```powershell
py -3 tools\register_template_to_individual.py `
  --fixed-t1 "E:\卷在哪里\生医工竞赛\T1图像\IXI002-Guys-0828-T1.nii.gz" `
  --template-t1 "mri\T1.nii" `
  --overlay "sample_data\aal.nii" `
  --label "sample_data\aal.nii" `
  --out-dir "registered\IXI002_AAL"
```

注意：AAL 原始文件更接近 MNI/AAL 空间，不一定和 `mri\T1.nii` 完全同空间。最严谨做法是使用与 AAL 同空间的 MNI T1 作为 `--template-t1`，再把 AAL 变换到个体空间。

## 质控

配准完成后不要只看文件是否能加载，至少检查：

- 头颅外轮廓是否贴合
- 中线是否基本一致
- 脑室位置是否大致吻合
- 电场是否大量落在脑外

如果这些不合理，需要换更好的模板、做去颅骨/偏置场校正，或使用 ANTs 非线性配准。
